Crée par : Phoenix C
Tools : ChatGPT Framework

Description : ChatGPT Framework est un framework pour utiliser toutes les possibilités de l'AI ChatGPT 
    d'Opanai. Il nécessite une clé D'API valid. Si vous n'en avez pas : 
    1 : Rendez-vous sur https://openai.com et créez-vous un compte. Cliquez sur View API key puis sur créer une nouvelle API.
    2 : Copiez la et rentrez la dans le programme quand c'est demandé.

    Le programme créera un fichier API.txt et verifiera son existence à chaque fois. Ce qui a pour avantage :
    Si vous supprimez le fichier, le programme le recréera. Il vous suffira de retapez l'API.
    Cela permet aussi de ne pas taper à chaque utilisation l'API.
